# N8N-test
repositorio para realizar pruebas con N8N, de manera automatizada

Aqui vamos a aprender a como instalar y desplegar la infraestructura necesaria para poder implementar N8N y usar la herramienta como tal

## notas 
Para instancias de EC2 es necesario adquirir una elastic ip si se quiere usar integraciones con whatsapp, ya que no genera codigo QR
se anexa bootstrap para instancias EC2, se instala lo necesario para usar el docker-compose.yml
